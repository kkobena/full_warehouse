Guide d'utilisation - Rotation du Stock
Qu'est-ce que la rotation du stock ?

La rotation du stock mesure la vitesse à laquelle un produit se vend et se renouvelle dans votre pharmacie. Une bonne gestion de la rotation permet d'optimiser la trésorerie, de réduire les stocks dormants et d'éviter les péremptions.

Classification ABC de rotation
Classe A - Forte rotation (z ≥ 1.96)

Caractéristiques : Produits qui se vendent rapidement et régulièrement

Actions :

  • Maintenir un stock suffisant pour éviter les ruptures
  • Commander fréquemment avec des quantités adaptées
  • Surveiller quotidiennement les niveaux de stock
  • Négocier de bons prix (volumes importants)
Classe B - Rotation moyenne (1.65 ≤ z < 1.96)

Caractéristiques : Produits avec une vente régulière mais modérée

Actions :

  • Commandes hebdomadaires ou bimensuelles
  • Stock de sécurité modéré
  • Surveillance hebdomadaire
Classe C - Faible rotation (z < 1.65)

Caractéristiques : Produits à vente lente ou occasionnelle

Actions :

  • Réduire les quantités en stock
  • Commander à la demande si possible
  • Surveiller les dates de péremption
  • Envisager promotions ou déréférencement
Formules et métriques clés

Taux de rotation annuel

Formule : Quantité vendue (12 mois) / Stock moyen
Interprétation :

  • ≥ 6x/an : Excellente rotation (vert)
  • 3-6x/an : Bonne rotation (bleu)
  • 1-3x/an : Rotation lente (orange)
  • < 1x/an : Stock dormant (rouge)

Jours en stock (Days of Supply)

Formule : (Stock actuel / Vente moyenne quotidienne) = Nombre de jours avant épuisement
Interprétation :

  • ≤ 30 jours : Stock optimal (vert)
  • 30-60 jours : Stock acceptable (bleu)
  • 60-90 jours : Stock excessif (orange)
  • > 90 jours : Stock dormant - risque de péremption (rouge)

Valeur du stock

Formule : Quantité en stock × Prix d'achat unitaire HT
Utilité : Calculer le capital immobilisé dans les stocks

Classification ABC (Score Z)

La classification utilise un score statistique (Z-score) basé sur la quantité vendue et sa variabilité pour classer les produits de manière objective :

  • z ≥ 1.96 : Classe A (forte rotation, top 2.5%)
  • 1.65 ≤ z < 1.96 : Classe B (rotation moyenne)
  • z < 1.65 : Classe C (faible rotation)
Comment exploiter cette analyse
Indicateurs d'alerte :
  • Produit classe A avec > 60 jours en stock : Baisse de la demande ? Ajuster les commandes
  • Produit classe C avec rotation < 1x/an : Envisager le déréférencement
  • Stock élevé + faible CA 30j : Risque de péremption, organiser une promotion
  • Taux rotation < 1 sur produits chers : Capital immobilisé inutilement
Exemples pratiques

✓ Cas 1 : Produit rotation 8x/an, 25 jours en stock

Analyse : Excellente rotation, stock optimal. Continuer la gestion actuelle. Maintenir un stock de sécurité pour éviter les ruptures.

✓ Cas 2 : Produit rotation 0.5x/an, 120 jours en stock

Analyse : Stock dormant. Actions : Vérifier date de péremption, organiser une promotion, réduire drastiquement les commandes ou déréférencer.

✓ Cas 3 : CA 12m élevé mais rotation faible

Analyse : Produit cher mais à rotation lente. Optimiser les quantités commandées pour libérer de la trésorerie sans impacter les ventes.

Bonnes pratiques
  • Analyser ce rapport mensuellement pour détecter les tendances
  • Croiser avec l'analyse ABC Pareto pour une vision globale
  • Adapter les quantités de commande selon le taux de rotation
  • Utiliser le filtre "Produits à faible rotation" pour identifier les stocks dormants
  • Privilégier les commandes fréquentes en petites quantités pour les produits classe A
