Guide d'utilisation - Prévisions de Ventes
Qu'est-ce que la prévision de ventes ?

La prévision de ventes utilise des modèles statistiques pour estimer le chiffre d'affaires futur en se basant sur l'historique des ventes. Cela permet d'anticiper les besoins, d'optimiser les stocks et de planifier la trésorerie.

Méthodes de prévision disponibles

1. Régression Linéaire (par défaut)

Principe : Calcule une droite de tendance à partir des données historiques
Formule : y = ax + b (où x = temps, y = CA prévu)

Utilisez-la si : Vos ventes suivent une tendance stable (croissance ou décroissance régulière)

2. Moyenne Mobile

Principe : Fait la moyenne des N derniers mois pour lisser les variations
Formule : CA prévu = Moyenne (CA des N derniers mois)

Utilisez-la si : Vos ventes sont relativement stables sans tendance marquée

3. Saisonnier (Recommandé si saisonnalité détectée)

Principe : Intègre les variations saisonnières (grippes en hiver, allergies au printemps, etc.)
Formule : CA prévu = Tendance × Coefficient saisonnier du mois

Utilisez-la si : L'alerte "Saisonnalité détectée" s'affiche (variations importantes selon les mois)

Comprendre les métriques

CA Prévu (Forecasted CA)

Estimation du chiffre d'affaires pour la période future selon la méthode sélectionnée.

Intervalle de Confiance (95%)

Limite Basse : CA minimum probable (scénario pessimiste)
Limite Haute : CA maximum probable (scénario optimiste)
Interprétation : Vous avez 95% de chances que le CA réel soit dans cet intervalle

Précision du Modèle (Model Accuracy)

Formule : 100% - MAPE (Mean Absolute Percentage Error)
Interprétation :

  • ≥ 80% : Excellent (prévisions très fiables)
  • 60-80% : Bon (prévisions fiables)
  • < 60% : Faible (utiliser avec prudence)

Croissance Mensuelle Moyenne

Formule : ((CA mois N / CA mois N-1) - 1) × 100
Indique si votre pharmacie est en croissance (+) ou en déclin (-)

Points de Données Utilisés

Nombre de mois d'historique utilisés pour calculer les prévisions. Plus ce nombre est élevé, plus la prévision est robuste.

Comprendre la saisonnalité
Saisonnalité détectée :

Le système détecte automatiquement si vos ventes varient significativement selon les mois. Par exemple, une pharmacie peut avoir :

  • Mois pic : Décembre-Janvier (grippes, virus hivernaux)
  • Mois creux : Juillet-Août (départs en vacances)

Si l'alerte s'affiche, utilisez la méthode "Saisonnier" pour des prévisions plus précises.

Comment exploiter ces prévisions

✓ Planification des stocks

Utilisez les prévisions 3-6 mois pour anticiper les besoins en produits. Augmentez les stocks avant les mois de forte demande prévue.

✓ Gestion de la trésorerie

Les prévisions 6-12 mois permettent d'anticiper les besoins de financement et de planifier les investissements (rénovation, nouveau matériel, etc.).

✓ Négociation avec les fournisseurs

Si une forte croissance est prévue, négociez des contrats-cadres ou des remises sur volumes avec vos fournisseurs principaux.

✓ Détection d'anomalies

Si le CA réel s'écarte fortement de la prévision (hors intervalle de confiance), analysez les causes : nouvelle concurrence, changement réglementaire, etc.

Exemples pratiques

Cas 1 : Prévision 3M = 15M FCFA, Précision 85%

Action : Planifiez vos commandes pour maintenir un stock de 3 mois. Fiabilité élevée, vous pouvez vous baser sur cette prévision en toute confiance.

Cas 2 : Saisonnalité détectée, mois pic = Décembre

Action : Augmentez vos stocks en novembre. Prévoyez du personnel supplémentaire en décembre. Utilisez la méthode "Saisonnier" pour affiner.

Cas 3 : Précision du modèle < 60%

Action : Prévisions peu fiables. Vos ventes sont probablement trop erratiques. Basez-vous sur l'intervalle de confiance et préparez un plan B.

Conseils d'utilisation
  • Comparez les prévisions des 3 méthodes pour identifier la plus pertinente
  • Mettez à jour les prévisions mensuellement pour intégrer les dernières données
  • En cas de changement majeur (rénovation, concurrence), recalculez les prévisions
  • Utilisez la prévision 3M pour les stocks, 6M pour la trésorerie, 12M pour la stratégie
  • L'intervalle de confiance est votre marge de sécurité : prévoyez le scénario pessimiste
