Information :

Le modèle de réapprovisionnement détermine comment sont calculés les seuils de stock minimum (qtySeuilMini) et les quantités de réapprovisionnement (qtyAppro) pour vos produits.

Cette configuration s'applique globalement à tous les produits lors du calcul automatique quotidien (entre 9h et 11h).

Quand utiliser chaque modèle ?
Modèle Classique

Recommandé si :

  • Vous débutez avec le système
  • Vos produits ont des ventes régulières
  • Vous n'avez pas le temps de configurer SEMOIS
  • Vous voulez un système simple et automatique
Modèle SEMOIS

Recommandé si :

  • Vous avez des produits avec des criticités différentes
  • Vous voulez optimiser vos stocks selon l'importance des produits
  • Vous gérez des produits à forte saisonnalité
  • Vous voulez minimiser les ruptures sur les produits critiques
Calcul automatique : Le réapprovisionnement est recalculé automatiquement tous les jours entre 9h et 11h. Le modèle sélectionné sera appliqué lors du prochain calcul.
