Guide d'utilisation - Analyse ABC Pareto
Qu'est-ce que l'analyse ABC Pareto ?

L'analyse ABC Pareto (règle 80/20) est une méthode de classification qui permet d'identifier les produits les plus rentables. Elle repose sur le principe que 20% des produits génèrent environ 80% du chiffre d'affaires.

Classification des produits
Classe A - Produits stratégiques

Représentent : ~20% des produits = 80% du CA cumulé

Actions :

  • Surveillance quotidienne des stocks
  • Négociation prioritaire avec fournisseurs
  • Éviter les ruptures de stock à tout prix
  • Prévoir des stocks de sécurité élevés
Classe B - Produits intermédiaires

Représentent : ~30% des produits = 15% du CA (80-95% cumulé)

Actions :

  • Surveillance hebdomadaire des stocks
  • Gestion optimisée des commandes
  • Maintenir un stock de sécurité modéré
Classe C - Produits occasionnels

Représentent : ~50% des produits = 5% du CA (95-100% cumulé)

Actions :

  • Commandes à la demande
  • Réduire les stocks dormants
  • Envisager le déréférencement si rotation trop faible
  • Stock minimal ou sur commande uniquement
Comprendre les métriques

Rang

Position du produit dans le classement par CA décroissant (1 = meilleur contributeur).

CA Total

Chiffre d'affaires généré par le produit sur la période analysée.

Contribution %

Formule : (CA du produit / CA global) × 100
Part du produit dans le CA total de la pharmacie.

CA Cumulé %

Formule : Somme des contributions du rang 1 jusqu'au rang actuel
Permet d'identifier les seuils 80%, 95% et 100% pour la classification ABC.

Comment exploiter cette analyse
Points d'attention :
  • Concentrez 80% de votre attention sur les produits de classe A
  • Surveillez les ruptures de stock sur la classe A (impact majeur sur le CA)
  • Optimisez la trésorerie en réduisant le stock des produits classe C
  • Renégociez les conditions d'achat avec vos fournisseurs pour les produits classe A
  • Utilisez le filtre "Top 50 Contributeurs" pour une vue focalisée
Exemples pratiques

✓ Cas 1 : Produit classe A en rupture

Action immédiate : Commander en urgence, proposer une alternative au client, augmenter le stock de sécurité pour éviter toute rupture future.

✓ Cas 2 : Produit classe C avec stock élevé

Action : Organiser une promotion pour écouler le stock, réduire la quantité commandée, voire envisager le déréférencement.

✓ Cas 3 : Négociation fournisseur

Stratégie : Utilisez l'analyse pour négocier de meilleurs prix sur les produits classe A (volumes importants = pouvoir de négociation).
