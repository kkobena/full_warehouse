Guide - Analyse de Rentabilité (Matrice BCG)
Qu'est-ce que la Matrice BCG ?

La Matrice BCG (Boston Consulting Group) est un outil stratégique qui classe vos produits selon deux critères : la rentabilité (taux de marge) et la dynamique commerciale (rotation). Elle permet d'optimiser votre portefeuille produits et de prendre des décisions stratégiques sur chaque catégorie.

Les 4 catégories de la Matrice BCG
STARS (Vedettes)

Caractéristiques : Marge élevée (≥ 20%) + Rotation élevée (≥ 4x/an)
Interprétation : Produits très rentables qui se vendent bien

Actions stratégiques :

  • Maximiser : Ces produits sont vos champions, investissez dans leur promotion
  • Maintenir des stocks élevés pour éviter les ruptures
  • Fidéliser les clients sur ces produits (conseils, disponibilité)
  • Surveiller la concurrence et ajuster les prix si nécessaire
CASH COWS (Vaches à lait)

Caractéristiques : Marge élevée (≥ 20%) + Rotation faible (< 4x/an)
Interprétation : Produits très rentables mais se vendent lentement

Actions stratégiques :

  • Optimiser : Produits rentables mais à faible rotation
  • Réduire les stocks (immobilisation de trésorerie)
  • Vendre sur commande ou en petites quantités
  • Maintenir les prix (bonne marge) mais sans sur-stocker
  • Envisager des promotions ciblées pour accélérer la vente
QUESTION MARKS (Dilemmes)

Caractéristiques : Marge faible (< 20%) + Rotation élevée (≥ 4x/an)
Interprétation : Produits qui se vendent bien mais peu rentables

Actions stratégiques :

  • Améliorer : Volume élevé mais faible rentabilité
  • Négocier de meilleurs prix d'achat avec les fournisseurs
  • Ajuster les prix de vente (si possible) pour augmenter la marge
  • Analyser les coûts cachés (transport, stockage, casse)
  • Décision stratégique : transformer en Star ou abandonner ?
DOGS (Poids morts)

Caractéristiques : Marge faible (< 20%) + Rotation faible (< 4x/an)
Interprétation : Produits peu rentables et qui se vendent mal

Actions stratégiques :

  • Éliminer ou transformer : Ces produits coûtent plus qu'ils ne rapportent
  • Déréférencer progressivement (sauf obligation réglementaire)
  • Liquider les stocks (promotions agressives)
  • Commander uniquement à la demande client
  • Libérer de l'espace de stockage et de la trésorerie
Comprendre les métriques

Taux de Marge (%)

Formule : ((Prix de vente HT - Prix d'achat HT) / Prix de vente HT) × 100
Seuils BCG :

  • ≥ 30% : Excellente marge (vert)
  • 20-30% : Bonne marge (bleu)
  • 10-20% : Marge faible (orange)
  • < 10% : Marge très faible (rouge) - Alerte !

Taux de Rotation Annuel

Formule : Quantité vendue (12 mois) / Stock moyen
Seuils BCG :

  • ≥ 4x/an : Rotation élevée (dynamique commerciale forte)
  • < 4x/an : Rotation faible (vente lente)

Marge Brute

Formule : CA - Coût d'achat
Utilité : Montre le profit réel généré par le produit en FCFA

Comment exploiter cette analyse
Stratégies par catégorie :
  • Stars : Investir et promouvoir - Ce sont vos produits d'or
  • Cash Cows : Optimiser les stocks - Rentables mais lents
  • Question Marks : Améliorer la marge ou abandonner - Décision stratégique
  • Dogs : Déréférencer - Libérer trésorerie et espace
Exemples de décisions concrètes

✓ Cas 1 : Star avec marge 35%, rotation 8x/an

Action : Produit champion. Augmentez le stock de sécurité, mettez-le en avant dans la pharmacie, formez votre équipe pour le conseiller activement.

✓ Cas 2 : Cash Cow avec marge 28%, rotation 2x/an

Action : Réduisez le stock (commandez 1x/trimestre au lieu de mensuel), maintenez le prix (bonne marge), surveillez la date de péremption.

✓ Cas 3 : Question Mark avec marge 12%, rotation 6x/an

Action : Volume important mais marge faible. Négociez avec le fournisseur pour baisser le prix d'achat de 5-10%. Si refus, augmentez légèrement le prix de vente.

✓ Cas 4 : Dog avec marge 8%, rotation 0.5x/an

Action : Produit non rentable. Promotion pour liquider le stock existant, puis arrêt du référencement (sauf si obligatoire). Récupérez votre trésorerie.

Portefeuille idéal

Un portefeuille équilibré devrait viser :

  • 40-50% de Stars : Moteurs de croissance et de profit
  • 20-30% de Cash Cows : Générateurs de marge stable
  • 10-20% de Question Marks : Potentiel d'amélioration
  • < 10% de Dogs : À éliminer progressivement
Signal d'alerte :

Si vous avez > 30% de Dogs dans votre portefeuille, votre rentabilité globale est menacée. Priorisez le déréférencement et concentrez-vous sur les Stars et Cash Cows.

Utiliser les filtres efficacement
  • Marges faibles (< 10%) : Identifiez rapidement les produits problématiques
  • Par catégorie BCG : Analysez chaque quadrant séparément
  • Par famille : Comparez la rentabilité entre catégories de produits
Fréquence de révision :

Analysez cette matrice trimestriellement. Les produits peuvent migrer d'une catégorie à l'autre selon les saisons, la concurrence ou les changements de marché. Une Star peut devenir Question Mark si la marge baisse (nouvelle concurrence par exemple).

Actions immédiates recommandées
  1. Cliquez sur "Marges faibles" pour voir tous les produits à < 10% de marge
  2. Filtrez par "Dogs" et établissez un plan de déréférencement sur 3-6 mois
  3. Filtrez par "Stars" et assurez-vous qu'ils ne sont jamais en rupture de stock
  4. Exportez le PDF et partagez-le avec votre équipe pour définir les priorités
