Guide d'utilisation
Formules et Métriques Clés

VMM - Ventes Mensuelles Moyennes

Formule : Moyenne pondérée des ventes sur N mois (par défaut 6 mois)
Pondération : Les mois récents comptent plus que les anciens
Exemple : Mois M-1 = poids 6, M-2 = poids 5, ..., M-6 = poids 1

Interprétation : Le VMM représente votre demande mensuelle typique. Un VMM de 50 signifie que vous vendez environ 50 unités par mois de ce produit.

Marge de Sécurité

Formule : VMM × (Délai livraison × Coefficient sécurité / 30) × Facteur saisonnier
Coefficient sécurité : Varie selon la classe de criticité (0.2 à 1.5)
Facteur saisonnier : Ajustement pour périodes spéciales (grippe, etc.)

Rôle : La marge de sécurité évite les ruptures de stock pendant le délai de livraison. Elle protège contre les variations de demande et les retards fournisseur.

Stock Objectif

Formule : VMM + Marge de Sécurité
Option péremption : Si activée, limité à MAX(Stock Obj, VMM × 3)

Objectif : Le niveau de stock idéal à maintenir. Il couvre votre demande mensuelle tout en assurant un matelas de sécurité.

Quantité à Commander

Formule : MAX(0, Stock Objectif - Stock Actuel)
Note : Toujours positif ou zéro (pas de commande si surstock)

Classification par Criticité
Classe A+ - Produits Vitaux

Exemples : Insuline, Adrénaline, Ventoline, Atropine

Actions :

  • Stock toujours disponible, rupture interdite
  • Surveillance quotidienne recommandée
Classe A - Forte Rotation

Caractéristiques : Top 20% des ventes, forte demande régulière

Actions :

  • Commandes fréquentes, stock suffisant
  • Suivi hebdomadaire, éviter ruptures
  • Négocier prix (volumes importants)
Classe B - Rotation Moyenne

Caractéristiques : Ventes régulières mais modérées (classe par défaut)

Actions :

  • Commandes hebdomadaires/bimensuelles
  • Stock de sécurité modéré
Classe C - Faible Rotation

Caractéristiques : VMM < 5 unités/mois, vente occasionnelle

Actions :

  • Réduire les stocks, commander à la demande
  • Surveiller dates de péremption
Classe D - Très Faible Rotation

Caractéristiques : VMM < 2 unités/mois ou nul

Actions :

  • Stock minimal, commande sur demande uniquement
  • Envisager déréférencement
Niveaux d'Urgence
URGENT Stock Actuel < Marge de Sécurité

⚠️ Risque de rupture imminent ! Commander en priorité absolue.

NORMAL Stock Actuel < Stock Objectif

Réapprovisionner lors de la prochaine commande.

OK Stock Actuel ≥ Stock Objectif

Stock optimal, pas de commande nécessaire.

Comment Utiliser SEMOIS
Workflow Recommandé :
  1. Configuration initiale :
    • Utiliser "Configuration en masse" pour classer tous vos produits
    • Ajuster les classes A+, A, C, D selon analyse ABC
    • Configurer délais de livraison par fournisseur
  2. Commandes quotidiennes :
    • Filtrer par "URGENT" → Commander immédiatement
    • Filtrer par classe A/A+ → Vérifier stock quotidiennement
    • Grouper commandes classe B/C par fournisseur
  3. Ajustements saisonniers :
    • Augmenter facteur saisonnier pour produits hiver (grippe, etc.)
    • Activer "Limite péremption" pour produits DLU courte
  4. Surveillance :
    • Vérifier cohérence VMM vs réalité tous les mois
    • Détecter anomalies (VMM élevé mais stock toujours plein = baisse demande)
Exemples Pratiques

✓ Cas 1 : Insuline (Classe A+)

VMM=15, Délai=5j, Coeff=1.5 → Marge=15×(5×1.5/30)=3.75≈4 → Stock Obj=15+4=19
Stock actuel=3 → URGENT (< marge 4) → Commander 16 unités immédiatement

✓ Cas 2 : Paracétamol (Classe A)

VMM=120, Délai=7j, Coeff=1.0 → Marge=120×(7×1.0/30)=28 → Stock Obj=148
Stock actuel=100 → NORMAL → Commander 48 unités prochaine commande

✓ Cas 3 : Produit rare (Classe C)

VMM=2, Délai=7j, Coeff=0.4 → Marge=2×(7×0.4/30)=0.18≈1 → Stock Obj=3
Stock actuel=5 → OK (surstock) → Pas de commande, attendre écoulement

Bonnes Pratiques
  • Laisser le système agréger les ventes quotidiennement (2h du matin)
  • Recalcul automatique SEMOIS tous les jours (3h du matin)
  • Ne pas modifier manuellement les agrégations mensuelles
  • Ajuster les classes de criticité tous les 3-6 mois selon évolution ABC
  • Utiliser facteur saisonnier pour événements prévisibles (grippe, allergies)
  • Activer limite péremption pour sérums, vaccins, produits DLU < 6 mois
