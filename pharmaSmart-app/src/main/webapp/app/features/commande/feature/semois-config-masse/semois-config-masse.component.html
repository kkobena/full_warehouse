Classification Auto (ABC)
Stratégie de Classification Automatique

Cette stratégie utilise l'analyse ABC de rotation existante pour classer automatiquement vos produits en classes SEMOIS. Elle se base sur le taux de rotation annuel et le z-score statistique.

Règles de classification proposées :
  • Classe A+ : Produits vitaux (à définir manuellement après import)
  • Classe A : ABC = A (forte rotation, z ≥ 1.96) ET rotation ≥ 6x/an
  • Classe B : ABC = B (rotation moyenne) OU rotation 3-6x/an
  • Classe C : ABC = C (faible rotation) ET rotation 1-3x/an
  • Classe D : Rotation < 1x/an ou VMM nul

Configuration Manuelle
Initialisation Globale des Produits

Initialise tous les produits actifs sans configuration SEMOIS avec des paramètres par défaut. Vous pourrez ensuite ajuster individuellement ou utiliser la classification ABC.

⚠️ Paramètres par défaut appliqués :
  • Classe de criticité : B (Rotation moyenne)
  • Coefficient de sécurité : 0.7
  • Nombre de mois historique : 6 mois
  • Délai de livraison : 7 jours
  • Facteur saisonnier : 1.0 (neutre)
  • Limite péremption : Non activée
