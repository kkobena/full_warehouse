Classification par Criticité
Classe A+ — Produits Vitaux

Exemples : Insuline, Adrénaline, Ventoline, Atropine

  • Stock toujours disponible, rupture interdite
  • Surveillance quotidienne recommandée
Classe A — Forte Rotation

Caractéristiques : Top 20% des ventes, forte demande régulière

  • Commandes fréquentes, stock suffisant
  • Suivi hebdomadaire, éviter ruptures
Classe B — Rotation Moyenne

Caractéristiques : Ventes régulières mais modérées (classe par défaut)

Classe C — Faible Rotation

Caractéristiques : VMM < 5 unités/mois, vente occasionnelle

Classe D — Très Faible Rotation

Caractéristiques : VMM < 2 unités/mois ou nul
